## Acerca de

Somos docentes, estudiantes y graduados de la **UNPA — UARG**, construyendo conocimiento y herramientas tecnológicas a través del código abierto. Esta organización en GitHub centraliza proyectos académicos, trabajos de cátedra e investigación aplicada.

**Misión:** Fomentar el aprendizaje colaborativo donde cada repositorio es un espacio de formación y cada contribución impulsa el crecimiento profesional.

## Guía de contribución

Si sos **estudiante**, **docente** o **egresado/a** de la UARG, podés participar siguiendo estos pasos:

1. **Solicitá acceso** a la organización a través de tu docente o el administrador.
2. **Creá un repositorio** o hacé fork de uno existente.
3. **Trabajá con buenas prácticas**: commits descriptivos, uso de branches y Pull Requests.
4. **Abrí un Issue** si encontrás un problema o necesitás orientación.

> **¿Es tu primer contacto con Git?** Consultá con tus docentes; hay guías y recursos disponibles para comenzar.

## Convención de commits

Utilizamos [Conventional Commits](https://www.conventionalcommits.org/) para mantener un historial claro y profesional:

| Prefijo | Uso |
|---------|-----|
| `feat:` | Nueva funcionalidad |
| `fix:` | Corrección de errores |
| `refactor:` | Mejora interna sin cambio de comportamiento |
| `docs:` | Cambios en documentación |
| `style:` | Formato, espaciado, puntuación (sin lógica) |

**Ejemplo:**

```bash
git commit -m "feat: agregar autenticación de usuarios"
```
