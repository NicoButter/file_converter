## 💡 ¿Sos estudiante o docente? ¡Este es tu espacio!

El software libre y la colaboración son los motores del cambio. Si formas parte de la UNPA UARG, te invitamos a:

*   **Explorar** los repositorios de tus materias 🧐
*   **Contactar** a tus profesores para saber cómo contribuir en los proyectos ✍️
*   **Compartir** tus propios proyectos y buscar colaboradores 🙌

### 🚀 ¿Cómo contribuir?

Si sos **estudiante**, **docente** o **egresado** de la UARG:

1. Pedí que te agreguen como miembro de la organización (contactá a tu docente o al administrador)
2. Creá o fork un repositorio
3. Trabajá usando **buenas prácticas** (commits claros, branches, Pull Requests)
4. Abrí un Issue si necesitás ayuda

**Estándar de Commits:** Usamos prefijos convencionales para mantener el historial organizado:
- `feat:` - Nuevas características o funcionalidades
- `refactor:` - Mejoras de código sin cambiar funcionalidad
- `fix:` - Corrección de bugs
- `docs:` - Cambios en documentación
- `style:` - Cambios de formato (espacios, puntuación, etc.)

Ejemplo: `git commit -m "feat: agregar autenticación de usuarios"`

**¿No sabes por dónde empezar con Git y GitHub?** No te preocupes, todos empezamos alguna vez. Consultá con tus docentes por recursos y guías para dar tus primeros pasos.
